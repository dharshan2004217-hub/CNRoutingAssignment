## Project Description
This repository includes Cisco Packet Tracer Simulations of Dynamic Routing Protocols.

## Files Included

1. RIP_Network.pkt
   - Configured RIP routing
   To check network connectivity using ping

2. EIGRP_Network.pkt
   - Configured EIGRP routing
   The RIP protocol was removed and replaced with EIGRP.The old RIP protocol was replaced by EIGRP.
   - Verified connectivity

3. Static_Network.pkt
   - Base network using static routing configuration
   The initial test of the topology is performed with this.This is used for the initial topology test.

## Testing
Ping command was used to test all networks between the PCs and the routers.
